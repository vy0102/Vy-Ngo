# Vy Ngo — Portfolio Website

A two-page personal portfolio website built with HTML, CSS, and JavaScript.

## Pages

- **index.html** — Cinematic hero landing page with animated headline, stats strip, and navigation
- **content.html** — About page with photo, education, technical skills, projects, work experience, and contact

## How to Deploy on GitHub Pages

1. Create a new repository on GitHub (e.g. `vy-portfolio`)
2. Upload all files to the repository:
   - `index.html`
   - `content.html`
   - Your photo file (e.g. `ChatGPT_Image_May_28__2026__01_41_08_PM.png`)
   - Your video file (e.g. `video_mp4.mp4`)
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at `https://yourusername.github.io/vy-portfolio`

## File Structure

```
/
├── index.html          ← Home / landing page
├── content.html        ← About page
├── video_mp4.mp4       ← Background video (upload yours)
├── ChatGPT_Image_...   ← Your portrait photo (upload yours)
└── README.md
```

## Notes

- Make sure your **photo filename** in `content.html` matches exactly what you upload to GitHub
- Make sure your **video filename** in `index.html` matches exactly what you upload to GitHub
- Both files use Google Fonts (Cormorant Garamond + Jost) — internet connection required to load fonts
